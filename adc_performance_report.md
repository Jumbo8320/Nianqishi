# ADC 动态性能测试结果分析

## 测试说明

本次测试使用 FPGA 内部新增的 raw ADC ILA 抓取 `rx_tdata_a[63:0]`，每个 JESD user clock 周期包含 4 个 ADC 样点。MATLAB 后处理将 64 bit payload 解包为连续 14 bit ADC 样点后进行 FFT 分析。

关键参数：

| 项目 | 数值 |
|---|---:|
| ADC 等效采样率 | 245.76 MSPS |
| 每次抓取 ILA beat 数 | 8192 |
| 解包后样点数 | 32768 |
| FFT 分辨率 | 7.5 kHz |
| 输入频率范围 | 1 MHz ~ 80 MHz |
| FFT 主峰保护带 | guardBins = 20 |

说明：本文不分析 Amplitude，仅分析 SNR、SFDR、SINAD、THD、ENOB 等动态指标。

## 指标趋势分析

### SNR

SNR 从低频到高频整体呈下降趋势：

| 频段 | SNR 表现 |
|---|---|
| 1 MHz ~ 10 MHz | 约 59.5 dB ~ 60.1 dB，表现最好 |
| 15 MHz ~ 40 MHz | 约 55.1 dB ~ 58.6 dB，中频段逐步下降 |
| 45 MHz ~ 80 MHz | 约 50.2 dB ~ 54.4 dB，高频段进一步下降 |

1 MHz 时 SNR 最高，为 60.06 dB；80 MHz 时 SNR 降至 50.17 dB。整体下降约 9.9 dB，说明随着输入频率升高，系统噪声、采样时钟抖动、模拟前端带宽和高频损耗等因素开始明显影响有效信噪比。

### SFDR

SFDR 在 1 MHz 处最好，为 66.73 dBc。随后随频率升高逐步下降，在 55 MHz ~ 65 MHz 附近达到最低：

| 频率 | SFDR |
|---:|---:|
| 1 MHz | 66.73 dBc |
| 10 MHz | 61.92 dBc |
| 30 MHz | 56.40 dBc |
| 50 MHz | 49.25 dBc |
| 60 MHz | 47.33 dBc |
| 80 MHz | 56.74 dBc |

SFDR 在 60 MHz 附近最低，之后 70 MHz、75 MHz、80 MHz 又有所回升。这说明限制 SFDR 的杂散源并不是单纯随频率单调恶化，可能与谐波折叠、前端频响、采样镜像或特定频段的 spur 分布有关。

### SINAD 与 ENOB

SINAD 和 ENOB 的趋势基本一致。低频段性能最好，高频段下降明显。

| 输入频率 | SINAD | ENOB |
|---:|---:|---:|
| 1 MHz | 59.14 dB | 9.53 bits |
| 10 MHz | 57.03 dB | 9.18 bits |
| 30 MHz | 52.79 dB | 8.48 bits |
| 50 MHz | 47.11 dB | 7.53 bits |
| 65 MHz | 46.00 dB | 7.35 bits |
| 80 MHz | 48.81 dB | 7.81 bits |

ENOB 从 1 MHz 的 9.53 bits 降到 65 MHz 附近的 7.35 bits，下降约 2.18 bits。说明在高输入频率下，系统的有效动态范围下降较明显。

### THD

THD 随输入频率升高总体变差，在 60 MHz ~ 65 MHz 附近最差：

| 输入频率 | THD |
|---:|---:|
| 1 MHz | -66.30 dBc |
| 10 MHz | -60.70 dBc |
| 30 MHz | -55.01 dBc |
| 50 MHz | -48.44 dBc |
| 60 MHz | -47.01 dBc |
| 65 MHz | -47.31 dBc |
| 80 MHz | -54.51 dBc |

THD 越负越好。低频段 THD 较好，说明低频输入下非线性失真较小。中高频段 THD 逐步恶化，说明模拟前端、ADC 输入网络或时钟相关失真在高频下更明显。80 MHz 处 THD 回升到 -54.51 dBc，表现好于 50 MHz ~ 65 MHz 区间。

## 频段划分评价

| 频段 | 综合评价 |
|---|---|
| 1 MHz ~ 10 MHz | 性能最好，SNR 接近 60 dB，ENOB 大于 9 bits，SFDR 大于 61 dBc |
| 15 MHz ~ 35 MHz | 性能开始下降，但整体仍较稳定，ENOB 约 8.3 ~ 8.8 bits |
| 40 MHz ~ 65 MHz | 性能下降最明显，SFDR、SINAD、THD 均进入较低区间 |
| 70 MHz ~ 80 MHz | SFDR 和 THD 有所回升，但 SNR 仍继续下降 |

## 结论

1. 该 ADC 采集链路在低频输入下表现较好，1 MHz 时 SNR 为 60.06 dB，SFDR 为 66.73 dBc，ENOB 为 9.53 bits。
2. 输入频率升高后，SNR、SINAD、ENOB 整体下降，说明高频输入下系统动态性能变差。
3. SFDR 在 55 MHz ~ 65 MHz 附近最低，最低约 47.33 dBc，是本次测试中杂散表现最差的频段。
4. THD 在中高频段明显恶化，60 MHz 附近约 -47 dBc，说明该频段非线性失真较强。
5. 70 MHz 以后 SFDR 和 THD 有一定回升，但 SNR 未恢复，说明高频端主要仍受噪声和前端带宽/时钟影响限制。

综合来看，本系统在 1 MHz ~ 10 MHz 输入范围内动态性能最好；在 40 MHz 以上性能开始明显下降；55 MHz ~ 65 MHz 是当前测试中最需要重点关注的频段。

## 后续建议

1. 对 55 MHz ~ 65 MHz 频段做更细步进测试，例如每 1 MHz 或 2 MHz 抓一次，确认 SFDR/THD 谷值位置。
2. 使用相干采样频率继续复测关键频点，减少 FFT 泄漏对 SFDR 判断的影响。
3. 对比不同输入幅度下的 THD 和 SFDR，判断失真是否主要来自前端非线性。
4. 同时观察 ADC 输入信号源频谱，排除信号源本身谐波或杂散对测试结果的影响。
